### Актуальность темы исследования <a name="par1.1"></a>
Доклад о мировом счастье - это исторический обзор состояния глобального счастья.
Первый Всемирный доклад о счастье был опубликован в апреле 2012 года в поддержку Совещания высокого уровня ООН по вопросам счастья и благополучия. С тех пор мир прошел долгий путь. Растущее счастье считается надлежащей мерой социального прогресса и целью государственной политики.
